This blog was initially built as a project of the [statistical computing](https://lcolladotor.github.io/jhustatcomputing/) class I'm taking, and was based on [Quarto](https://quarto.org/).

blog structure
- index.qmd -> homepage
- listings -> dir for all listing pages
	- work.qmd
	- personal.qmd
	- archive.qmd
- work -> dir for work-related articles
- personal -> dir for personal articles
- scripts -> homepage js plug-ins
- figs -> homepage figs

every sub directory should be named in this way: yymmdd_title
